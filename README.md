How to Run the Project

Follow these steps to set up and run the project on your local machine:

Create a Virtual Environment (VENV)

Inside the backend folder, create a virtual environment:

python -m venv venv

Activate the Virtual Environment
venv/Scripts/activate

Install All Dependencies
pip install -r requirements.txt

Configure Environment Variables

Create your .env file and make sure to configure all required variables for your machine.

Run the Project
python main.py

⚠️ Important
Make sure to create the database before running the project.
